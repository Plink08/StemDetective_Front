Waarom StemDetective?

In een democratie is transparantie essentieel. Kiezers geven een mandaat op basis van beloftes, maar de praktijk is vaak weerbarstig.

Met StemDetective maken we stemgedrag objectief inzichtelijk door data direct uit de bron te halen.

  • Data uit officieel Tweede Kamer archief
  • Koppeling met de live Tweede Kamer API
  • Analyse van de 250 meest recente moties
  • Objectieve AI-verwerking
